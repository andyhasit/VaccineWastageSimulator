Inputs

Annual births is the expected number of births in one year.

Doses per recipient is the number of doses each child should receive.

Target coverage is the expected or hoped for % of children fully immunized.

Sessions per week is the nominal number of immunization sessions per week at the location in question.

Vial size is the nominal number of doses per vial.

Charts

The first chart shows the expected number of sessions per year with 0, 1, 2, 3, ... doses administered (for each session size) given the expected number of doses administered per year and sessions per week.

The second chart shows the wastage rate for each session size given the vial size.
